There is a debian package repository for Lucid at http://gholt.github.com/swauth/debian/ so you can easily install Swauth on Lucid by:

  • Create /etc/apt/sources.list.d/swauth.list:

    deb http://gholt.github.com/swauth/debian lucid main

  • Then:

    sudo apt-get update
    sudo apt-get install python-swauth swauth-doc
